# Task Management Tool For Managing projects

This is an intuitive, modern web based system that helps manage tasks efficiently and promote teamwork. User experieence design elements such as a centrally position login user interface and interactive dashboarf area with data and visualizations help to achieve this.

## Features

* **Project Management:** Seamlessly create, amange and track multiple projects.
* **Task Management:** Ability to add, assign and manage specific tasks wihtin a project.
* **Team Coordination:** Ability to add and manage all the team members of the team who will be participating in different workflows.
* **Modern UI/UX:** Features a nicely styled, centrally positioned login interface, smooth responsive design and built in interactive data charts.

## Frameworks & Libraries Used

To achieve a responsive and visually appealing user experieence the following fronted technologies are utilized:

* **[Bootstrap](https://getbootstrap.com/):** Used for layout, responsive grid system and UI elements such as buttons, forms and text fields.
* **[Bootstrap Icons](https://icons.getbootstrap.com/):** Used for simple, scalable icon designs throughout the application.
* **[Chart.js](https://www.chartjs.org/):** Used to create interactive data driven charts displayed on the project dashboard.
* **[Vanta.js](https://www.vantajs.com/):** Used to create visually dynamic animated background effects displayed on the login user interface and other pages.

## Project Structure

The project follows an organization of individual components to maintain separation of concerns which promotes clean maintainable code:

* **Login Page:** To log into the page.
* **Dashboard page:** The start of the information and visualization page for data input and tracks.
* **Team Page:** Page for user to put in information related to the team member.
* **Project Page:** Page for project input and types to be put accrodingly.
* **Task Page:** Page for Task input and listing.
